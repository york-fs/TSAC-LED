This board is based on the design by Rootthecause, whos original design can be accessed [here](https://github.com/Rootthecause/HVI)

Changes made (Listed in order to conform to the CERN Open Hardware Licence):
- The two connectors as input has been changed to a single molex microfit plus connector
- The output connector has been replaced with a molex microfit plus equivalent
- The DC/DC converter has been changed to a 12V one, in order for a 12V output
- The board has been restructured and more silkscreen elements have been added
